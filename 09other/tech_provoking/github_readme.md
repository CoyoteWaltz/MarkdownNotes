## Status

https://github.com/anuraghazra/github-readme-stats

Get **dynamically** generated GitHub stats on your READMEs!

可以直接通过 api 来获得动态内容，跟着 readme 来配置就行

主题选用 gruvbox，当然也可以用 `bg_color=0,xxx,xxx` 设置渐变色，但感觉太花哨了，不喜欢
